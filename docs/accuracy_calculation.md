# Accuracy Calculation

Accuracy = (TP + TN) / (TP + TN + FP + FN) × 100 %

**100-hour outdoor field trial results (Multan, Pakistan):**
- True Positives (TP)   = 1,163
- True Negatives (TN)  = 2,910
- False Positives (FP)  = 47
- False Negatives (FN)  = 77

→ **Accuracy = 93.8 %**  
*(Source: Muxtronics field-trial log – see `utils/field_trials_log.txt`)*
